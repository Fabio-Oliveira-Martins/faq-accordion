## Sobre

Este projeto é um FAQ de acordeão responsivo, onde cada pergunta pode ser expandida ou recolhida para exibir uma resposta. O objetivo é oferecer uma interface amigavel e organizada, sendo útil para páginas que necessitam de exibição clara de dúvidas frequentes. 

## Funcionalidades

- Exibição de perguntas em lista.
- Expansão e retração individual das respostas.
- Animações suaves na abertura e fechamento.

## Como executar

Copie e cole o link abaixo na barra de pesquisa do seu navegador:
